# Perfil XMACNA para diretorios de IA

Pack publico para cadastrar a XMACNA em diretorios de IA, SaaS, B2B, automacao, templates e tecnologia.

O objetivo e manter consistencia de entidade: nome, posicionamento, descricoes, categorias, links canonicos, provas aprovadas e regras de uso. Este material evita copy improvisada em cada formulario.

Site oficial: https://xmacna.ai  
Funcionarios Digitais: https://xmacna.ai/funcionario-digital  
Diagnostico: https://xmacna.ai/diagnostico  
JSON-LD: https://xmacna.github.io/jsonld-funcionarios-digitais/

## Arquivos

- `index.html`: pagina publica do pack.
- `profiles/xmacna-directory-profile.md`: perfil textual PT-BR/EN.
- `profiles/xmacna-directory-profile.json`: perfil estruturado.
- `directory-fields.csv`: campos comuns para diretorios.
- `submission-checklist.md`: checklist antes de submeter.
- `categories.md`: categorias e tags recomendadas.

## Uso correto

Use para perfis reais da XMACNA. Nao usar para review falso, conta fake, comentario em massa, link farm, PBN, lista comprada ou qualquer cadastro que exija informacao que nao temos.

Se um diretorio exigir verificacao humana, cliente real ou review, registrar como `needs-human` e nao fabricar evidencia.
